# Predictive Maintenance for Factory Equipment

## Project Description
Predictive Maintenance for Factory Equipment is a machine learning and deep learning based system 
that predicts equipment failures before they occur. The system analyzes machine sensor data 
such as air temperature, process temperature, rotational speed, torque, and tool wear to monitor machine health. 
The goal is to reduce unexpected breakdowns and improve maintenance planning in factories.

## Features
- Predict machine failures using sensor data  
- Compare multiple machine learning models  
- Use deep learning for improved accuracy  
- Estimate machine health condition  
- Simple user interface built with Streamlit  

## Algorithms Used
- Support Vector Machine (SVM)
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Convolutional Neural Network (CNN)

## Technologies Used
- Python  
- Scikit-learn  
- TensorFlow / Keras  
- Pandas & NumPy  
- Streamlit  

## How to Run the Project
1. Clone the repository
2. Install required libraries

pip install -r requirements.txt

3. Run the application

streamlit run app.py


## Output
The system predicts the machine condition and displays whether maintenance is required. It also shows the prediction confidence and helps detect possible failure types.
